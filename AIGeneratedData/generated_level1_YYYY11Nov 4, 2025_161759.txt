Q1: What does the `print()` function do in Python, and how should text be formatted inside it?  
A1: The `print()` function displays text or numbers on the screen, and text must be enclosed in quotes.

Q2: How can you write a comment in Python, and what is its purpose?  
A2: Comments start with `#` and are ignored by Python; they help explain code.

Q3: What are the rules for naming variables in Python?  
A3: Variable names must begin with a letter or underscore, have no spaces, and are case-sensitive.

Q4: Name four common data types in Python and give an example of each.  
A4: Integers (`int`, e.g., 5), decimals (`float`, e.g., 3.14), text strings (`str`, e.g., "Hello"), and Booleans (`bool`, e.g., True).

Q5: Which arithmetic operation in Python always returns a float, and which one returns the quotient rounded down?  
A5: Division (`/`) always returns a float, and floor division (`//`) returns the quotient rounded down.

Q6: How do you access the first character of a string in Python?  
A6: By using indexing with `[0]`, since indexing starts at 0.

Q7: What function do you use to get user input, and what type of data does it return?  
A7: Use `input()`, and it returns data as a string.

Q8: How do you convert a string to an integer or a float in Python?  
A8: Use `int()` to convert to integer and `float()` to convert to float.

Q9: What are the Boolean values in Python, and what operators can compare values?  
A9: Boolean values are `True` and `False`; comparison operators include `==`, `!=`, `>`, `<`, `>=`, and `<=`.

Q10: What keywords do you use in Python to run different code based on multiple conditions?  
A10: Use `if`, `elif`, and `else`.

Q11: How does a `while` loop differ from a `for` loop in Python?  
A11: A `while` loop repeats code while a condition is true; a `for` loop iterates over items in a sequence.

Q12: Name three common list methods in Python.  
A12: `.append()`, `.remove()`, and `.sort()`.

Q13: What is the key difference between a list and a tuple?  
A13: Lists are changeable (mutable), while tuples are immutable (cannot be changed).

Q14: How do you define a function in Python that can return a value?  
A14: Use `def` to define the function and `return` to send back a value.

Q15: Which Python statement is used to include code from external modules?  
A15: Use `import`.

Q16: How can you display a string and a number together using `print()` without causing an error?  
A16: Use commas to separate them in `print()`, or convert the number to a string with `str()`.

Q17: What mode should you use with `open()` to write to a file, and how can you ensure the file is properly closed?  
A17: Use mode `'w'` to write, and use a `with` statement to automatically close the file.

Q18: What is the purpose of indentation in Python code?  
A18: Indentation defines code blocks, showing which statements belong together.

Q19: How do you get both the keys and values when looping through a dictionary?  
A19: Use the `.items()` method.

Q20: What command stops a loop early, and what command skips to the next iteration?  
A20: `break` stops a loop; `continue` skips to the next iteration.
